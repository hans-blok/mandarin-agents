# README: Templates voor thema-verwoorder

Deze folder bevat de output templates voor de agent "thema-verwoorder". Elke template beschrijft de structuur van de output die de agent per intent moet opleveren.

## Templates
- `thema-verwoorder.definieer-epic-structuur.template.md`: Voor het gestructureerd vastleggen van een epic volgens Safe-conventies.
- `thema-verwoorder.definieer-verbeter-voorstel.template.md`: Voor het voorstellen van verbeteringen aan bestaande epics of thematische structuren.
- `thema-verwoorder.definieer-thematische-scope.template.md`: Voor het afbakenen en documenteren van de thematische scope binnen de value stream.

## Gebruik
Vul de `{placeholders}` in met de relevante gegevens. De agent gebruikt deze templates om consistente, gestructureerde output te genereren per intent.
