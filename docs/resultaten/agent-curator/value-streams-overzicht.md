# Value Streams Overzicht

**Laatst bijgewerkt**: 2026-01-22  
**Beheerd door**: Agent Curator

Dit overzicht bevat alle geregistreerde value streams in de agent-services workspace. Value streams worden uitsluitend door mensen gedefinieerd en door Agent Curator geadministreerd.

---

---

## Toelichting

**agent-enablement**: De meta-stream voor agent-ontwikkeling zelf. Bevat agents zoals Agent Smeder, Agent Curator en Moeder die het agent-ecosysteem mogelijk maken. Agents: agent-smeder, agent-curator, moeder.

**architectuur-en-oplossingsontwerp**: Modellering en ontwerp van enterprise architectuur en IT-oplossingen. Omvat agents voor ArchiMate-modellering, C4-diagrammen, ADR's en andere architectuur-artefacten. Richt zich op het vastleggen van architecturale beslissingen, modellen en oplossingsontwerpen. Agents: archimate-modelleur, c4-modelleur, bedrijfsarchitect, converter-md-to-archimate.

**it-development**: Richt zich op software-ontwikkeling workflows, inclusief pipeline-executie, workflow-architectuur en ontwikkel-automatisering. Agents: pipeline-executor, workflow-architect, archimate-modeler.

**kennispublicatie**: Alle agents die kennis produceren, transformeren of publiceren. Van artikel-schrijvers tot publishers en presentatie-architecten. Agents: agent-publisher, artikel-schrijver, de-schrijver, essayist, heraut, presentatie-architect, vertaler.

**ondernemingsvorming**: Strategische laag voor organisatie- en systeem-principes. Definieert enterprise architecture, analyseert value streams, identificeert gaps en plateaus, en plant transformatie-werkpaketten. Agents: mandarin-ea, presentatie-architect.

**utility**: Cross-cutting agents die in meerdere streams worden gebruikt voor technische ondersteuning. Agents: python-expert.
